# Tech Stack

PHP (8.0) <br>
MySQL <br>
Bootrap <br>
Medoo ORM <br>

## Installation

Apache server or any other web server that supports php code

```bash
git clone https://github.com/adushaj2022/book-keeper.git
php composer.phar install
```
